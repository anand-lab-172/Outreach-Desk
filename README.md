# Outreach Desk

A lightweight outreach dashboard built for recruiters, HR outreach, and job applications.

Instead of manually copying emails and resumes every day, Outreach Desk lets you manage contacts, generate email drafts, and keep your resume just one click away.

> Built with pure HTML, CSS, and JavaScript — no backend required.

---

## ✨ Features

- 📇 Contact management
  - Add individual contacts
  - Bulk import contacts
  - Company tracking
  - Remove contacts

- 📧 One-click email drafts
  - Gmail compose support
  - Default mail client support
  - Dynamic placeholders
    - `{{name}}`
    - `{{role}}`

- 📋 Message Templates
  - Save custom email templates
  - Auto-fill recruiter name
  - Auto-fill job role

- 📊 Dashboard
  - Total Contacts
  - Pending Applications
  - Sent Applications
  - Daily Streak

- 📄 Resume Integration
  - Automatically uses the hosted resume from GitHub Pages
  - One-click resume access
  - Copy resume link instantly

- 💾 Local Storage
  - Contacts
  - Templates
  - Preferred email provider
  - Daily streak

## 📧 Email Template Variables

Use the following placeholders in your message template:

| Placeholder | Description |
|-------------|-------------|
| `{{name}}` | Recruiter's first name |
| `{{role}}` | Job title |

Example:

```
Hi {{name}},

I came across the {{role}} opening and would love to discuss the opportunity.
```

## 🎯 Why I Built This

As a Data Scientist actively applying for opportunities, I wanted a simple dashboard that could:

- keep recruiter contacts organized
- reduce repetitive copy-paste work
- generate email drafts quickly
- provide instant access to my latest resume

Instead of using a heavy CRM, I built a lightweight browser-based solution that works entirely on GitHub Pages.
